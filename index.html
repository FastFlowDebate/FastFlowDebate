Fast Flow

Welcome to FastFlow

FastFlow is a tool to help debaters leverage their research better in round. Debaters put enormous amounts of effort into cutting cards and creating blocks, and Fastflow is designed to make utilizing that preperation easier and more efficient.

FastFlow's source code is available on Github so if you have any bugs feel free to submit an issue or pull request.

Features

CardsBeta

FastFlow replaces folder style organization with tag based searching, simply attach tags to cards and search for cards how you want to!

SpeechWIP

Whereas Word only offers you a texteditor for writing speeches, FastFlow gives you a tool for presenting speeches.

Helping you ensure your speech is to time with smart time indicators, and with links that make sure any file you reference is only one click away.
